# PhotoFilter-Django-OpenCV
A photo filter web application using OpenCV with Django


#### Built With
- Html
- CSS
- Javascript
- Django
- OpenCV

#### Features
- No Filter
- Colorized 
- Gray
- Blurred
- Binary
- Invert Color
